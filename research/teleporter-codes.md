## Notes for Clone Design

- The teleporter code system is a memorable gameplay element worth preserving
- Consider generating fun/thematic 5-character codes for the clone
- Players in the original had to **memorize or write down** codes -- there was no in-game map
- For mobile, consider whether to keep this memorization mechanic or provide a discovered-codes list
- The code entry interface needs to work well with touch input
